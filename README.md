# Smart_Home_Central_Hub
A group project for SWDV 650 at Maryville University

# How to run:
git clone https://github.com/Saints-Squad-2/Smart_Home_Central_Hub.git

cd ./Smart_Home_Central_Hub

npm install

npm start OR node app.js

# How to run tests: 
npm test
